開始 GitHub 之旅 學習 Gemini 開發

GitHub 帳號申請與使用

全球最大的程式碼託管平台，開發者的履歷與協作中心。

1 帳號註冊流程

訪問官網

前往 github.com，點擊右上角的 "Sign up" 按鈕。

選擇註冊方式

推薦點選 Google 快速註冊。
或是手動輸入 Email、設定密碼並建立使用者名稱。

驗證與方案

完成機器人驗證與 Email 驗證碼確認。選擇 "Free" (免費) 個人方案即可開始使用。

2 建立第一個 Repository

新增專案

點擊右上角 "+" 號，選擇 New repository。

設定內容

  • Repository name: 輸入專案名稱 (如: my-first-website)
  • Public/Private: 選擇公開或私人
  • 勾選 Add a README file (建議新手勾選)

上傳檔案

建立後，點擊 Add file > Upload files，將你的網頁檔案拖曳上傳，並點擊 "Commit changes" 保存。

AI 賦能開發

利用 Gemini 設計教學網頁

學會「提示工程 (Prompt Engineering)」，讓 AI 成為你的資深工程師。

1. 定義角色與目標

告訴 Gemini 它是誰 (如：前端專家)，以及你要做什麼 (如：響應式個人履歷)。

2. 提供具體細節

包含技術棧 (HTML, Tailwind)、配色偏好、需要的區塊 (Hero, About, Contact)。

3. 迭代與修正

AI 第一次產出可能不完美。請具體指出問題 (如：字體太大、導覽列要固定) 讓它修改。

實戰：萬用 Prompt 範本

複製以下指令貼給 Gemini，立即獲得高品質網頁代碼。

# 角色 你是一位資深的網頁前端設計師。 # 任務 請幫我用 HTML5 與 Tailwind CSS (CDN版本) 製作一個單頁式網頁。 # 內容需求 1. 主題：[填入你的主題，例如：咖啡店介紹] 2. 區塊：導覽列、Hero區塊(大圖與標題)、特色介紹(3欄式卡片)、頁尾。 3. 風格：[填入風格，例如：極簡、溫暖色調]。 # 限制 所有 CSS 都要寫在 class 裡，確保 RWD 手機版也能正常顯示。

Gemini 思考邏輯

  • 分析用戶提供的結構需求
  • 選擇 Tailwind Utility Classes
  • 生成 HTML 結構
  • 優化手機版面配置 (Responsive)
