1. Gemini 環境導覽

Google 最強大的 AI 助手

為何選擇 Gemini？

  • ⚡ 生態系整合：直連 Docs, Gmail, Drive
  • 👀 多模態能力：看圖、讀 Code、聽聲音
  • 🚀 速度優勢：長文本處理能力強

介面速覽

(建議現場開啟 Gemini 網頁演示)

  • 左側邊欄：歷史對話與設定
  • 中央輸入區：文字、圖片上傳、麥克風
  • 回答區：按讚/修改/匯出功能

3. GitHub 帳號註冊

工程師的社群，你的作品集基地

註冊流程 (Sign Up)

  1. 進入 github.com
  2. 輸入 Email (建議用常用信箱)
  3. 設定密碼 (需符合強度)
  4. 關鍵：設定 Username
    (這會變成你的網址 ID，勿亂碼)
  5. 完成人機驗證 (Verify)

4. 分享你的網頁

GitHub Pages 部署實戰

Step 1: 建立倉庫 (Repository)

  • 點擊右上角 "+" -> New repository
  • 輸入專案名稱 (例: my-first-ai-web)
  • 務必選擇 Public (公開)
  • 勾選 "Add a README file"

Step 2: 上傳檔案

  • 點擊 "Add file" -> "Upload files"
  • 將 HTML 檔拖曳進去
  • ⚠️ 重要：主頁檔名必須是 index.html
  • 點擊 "Commit changes"

Step 3: 開啟 Pages 功能

  • 進入 Settings -> 左側選 Pages
  • Branch 設為 main / Folder 設為 root
  • 點擊 Save，等待約 1-2 分鐘
  • 獲得網址！
    username.github.io/project-name
